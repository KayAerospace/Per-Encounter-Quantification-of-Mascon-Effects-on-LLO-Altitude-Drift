# Per-Encounter-Quantification-of-Mascon-Effects-on-LLO-Altitude-Drift
Encounter-based framework for decomposing lunar mascon perturbations in low lunar orbits. Identifies discrete gravitational encounters via threshold-gating, maps per-encounter eccentricity-vector increments using Gauss VoP, and quantifies natural cancellation, establishing a foundation for mascon surfing station-keeping. Built using GRAIL GRGM900C.

The code supporting this study is archived on Zenodo at https://doi.org/10.5281/zenodo.19097158
